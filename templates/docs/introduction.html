    Authome 이란?

    개요

    프로그램의 사용자 관리를 목적으로 xncbf 가 만든 사이트입니다.
    본 사이트는 유저 관리 API를 제공하고 개발자는 API를 제공받아 사이트간의 통신을 통해 유저를 관리합니다
    Django 로 제작되었으며 데이터베이스를 제외한 모든 소스코드는 오픈소스형태로 제공됩니다.

    운영

    기부를 통해 운영되고 소스코드에 직,간접적으로 기여할 수 있습니다.

    이름

    Authome 은 Authorization을 줄인 Auth 에 발음이 비슷한 Awesome 을 합쳐 만든 단어입니다.

    기여하기

    버그리포트

    모든 사용자는 아래의 주소를 통해서 사용자 경험에 의거한 사이트 개선을 요구할 수 있습니다.
    주소 : https://github.com/xncbf/authome/issues

    Contribute

    버그리포트를 통해 요청된 개선사항이나 스스로 수정이 필요하다고 생각되는 부분을 pull request 를 통해 직접 참여할 수 있습니다.
